I want this mod to add a Satisfactory style storage dimension, where I can input items into an infinitely large void, and have access to them anywhere. However, only the player should be able to take the items out.
The mod should have these blocks:
- Network Controller
  - Requires SU to run
- Network Uplink
  - No SU
  - Allows inputting into the storage network
  - should act as an in only inventory.
  - Bound to a Network controller using the same system as Create's Stock network. 
- Accessor
  - Allows the player to access the storage network.
  - Has to be bound to a network.
  - Imitates a toolbox, where you can pin items in the GUI, and then the create toolbox UI on alt hold and everything else the toolbox does.
